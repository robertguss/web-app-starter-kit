# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-10

### Added
- Initial release of AI Starter Kit
- Next.js 16 with React 19 and App Router
- Convex real-time database integration
- Better Auth authentication system
  - Email/password authentication
  - Session management
  - Protected routes via middleware
- shadcn/ui component library (20+ components)
- Tailwind CSS 4 styling
- TypeScript strict mode configuration
- Vitest + convex-test testing setup
- Example Convex functions (queries, mutations, actions)
- Comprehensive documentation:
  - Quick Start Guide
  - Detailed Setup Guide
  - Architecture Overview
  - Development Guide
  - Deployment Guide
  - API Reference
  - Database Guide
  - Authentication Guide
  - Troubleshooting Guide
  - IDE Tools Documentation
- Contributing guidelines
- Code of Conduct
- MIT License
- GitHub issue and PR templates
- Example dashboard with charts and data tables
- Responsive sidebar navigation
- Dark mode support

### Developer Experience
- Hot module replacement with Turbo mode
- Parallel dev servers (frontend + backend)
- Automatic TypeScript generation from schema
- ESLint configuration
- Path aliases (@/components, @/lib, etc.)

### Documentation
- README with comprehensive project overview
- Step-by-step setup instructions
- Code examples and patterns
- Deployment guide for Vercel
- Testing patterns and best practices
